Anime & Movie API

This API provides access to trending, popular, and upcoming anime, movies, and TV shows. It leverages the Jikan API for anime data and the TMDB API for movie & TV show data. Integrate these endpoints into your applications to fetch information about anime, movies, and TV shows.

Features

Anime Endpoints

  • GET /trending/anime - Fetch currently trending anime series.
  • GET /popular/anime - Retrieve a list of popular anime series.
  • GET /upcoming/anime - Get information about upcoming anime releases.

Movie Endpoints

  • GET /trending/movies - Fetch currently trending movies.
  • GET /popular/movies - Retrieve a list of popular movies.
  • GET /upcoming/movies - Get information about upcoming movie releases.

TV Endpoints

  • GET /trending/tv - Fetch currently trending TV shows.
  • GET /popular/tv - Retrieve a list of popular TV shows.
