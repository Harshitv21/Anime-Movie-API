Anime & Movie API

This API provides access to trending, popular, and upcoming anime and movies. It leverages the Jikan API for anime data and the TMDB API for movie data. Integrate these endpoints into your applications to fetch anime and movie information.

Features

Anime Endpoints

  • GET /anime/trending - Fetch the currently trending anime series.
  • GET /anime/popular - Retrieve a list of popular anime series.
  • GET /anime/upcoming - Get information about upcoming anime releases.

Movie Endpoints

  • GET /movies/trending - Fetch the currently trending movies.
  • GET /movies/popular - Retrieve a list of popular movies.
  • GET /movies/upcoming - Get information about upcoming movie releases.
