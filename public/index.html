Anime & Movie API

Welcome to the Anime & Movie API, your go-to solution for accessing detailed and up-to-date information on anime, movies, and TV shows. Using data from the Jikan API for anime and the TMDB API for movies and TV shows, this API enables developers to build applications with dynamic media content.

Key Features

Anime Endpoints (Powered by Jikan API)

  • GET /trending/anime - Get the latest trending anime series.
  • GET /popular/anime - Retrieve a list of popular anime shows, updated frequently.
  • GET /upcoming/anime - Fetch details about upcoming anime releases.
  • GET /search/anime - Search for specific anime titles using various filters such as genre, score, and more.

Movie Endpoints (Powered by TMDB API)

  • GET /trending/movies - Retrieve a list of movies currently trending.
  • GET /popular/movies - Get a list of movies that are popular worldwide.
  • GET /upcoming/movies - Access information on upcoming movie releases.
  • GET /search/movies - Search for movies based on title, release date, and other criteria.

TV Show Endpoints (Powered by TMDB API)

  • GET /trending/tv - Discover trending TV shows from across the globe.
  • GET /popular/tv - Fetch popular TV shows from a wide variety of genres.
  • GET /search/tv - Search for TV shows by name, genre, or other criteria.
